ENGLISH — ENABLE DEVELOPER MODE (iOS 16+)

  1. On iPhone, go to Settings → Privacy & Security → Developer Mode.
  2. Toggle On → tap Restart when prompted.
  3. After reboot, confirm Turn On when iPhone asks.
  4. Return here and tap INSTALL if needed.

TIẾNG VIỆT — BẬT CHẾ ĐỘ NHÀ PHÁT TRIỂN (iOS 16+)

  1. Trên iPhone, vào Cài đặt → Quyền riêng tư & Bảo mật → Chế độ nhà phát triển.
  2. Gạt Bật → nhấn Khởi động lại khi được yêu cầu.
  3. Sau khi khởi động lại, xác nhận Bật khi iPhone hỏi.
  4. Quay lại trang này và nhấn INSTALL nếu cần.
